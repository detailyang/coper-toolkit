# Usage

```
NAME:
   coper-toolkit

USAGE:
   coper-toolkit [global options] command [command options] [arguments...]

VERSION:
   0.1.0

DESCRIPTION:
   personal copernicus toolkit to analyze something, use at your own risk:)

COMMANDS:
     parsedat         parse dat file
     parserev         parse rev file
     parseindex       parse index dir
     parsechainstate  parse chainstate
     evalscript       eval script
     disassemble      disassemble script
     help, h          Shows a list of commands or help for one command

GLOBAL OPTIONS:
   --help, -h     show help
   --version, -v  print the version
```
